# 🚀 AI Market Intelligence System (Agentic AI Powered)

An end-to-end AI-powered platform that transforms raw market data into
actionable investment decisions using a multi-agent reasoning system.

## 🧠 Problem

Retail investors are overwhelmed with fragmented data. Most platforms
provide information, not decisions.

## 💡 Solution

An Agentic AI system that: - Discovers opportunities - Generates
BUY/WATCH/AVOID decisions - Analyzes portfolio risk - Creates market
briefings

## ⚙️ Architecture

Frontend (Next.js) → API Layer → FastAPI → AI Engines

## 🤖 Agentic AI

Technical, Sentiment, Fundamental, Bull, Bear, and Judge agents
collaborate to produce decisions.

## 🔍 Features

-   AI Discovery Engine
-   Decision Engine
-   Portfolio Intelligence
-   Market Briefing

## ⚙️ Tech Stack

Frontend: Next.js\
Backend: FastAPI\
AI: Multi-agent LLM system

## 🚀 Run

Backend: uvicorn server:app --reload --port 8000

Frontend: cd et_ui npm install npm run dev

Open: http://localhost:3000

## ⭐ Note

We don't provide data --- we provide decisions.
